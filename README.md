## 게임 정보
> **코로나 바이러스로 살아남기**

### 맵
10×10 맵이며, 배경은 사람의 몸 내부이다.  
몸 속을 돌아다니며 코로나에 감염시키는 것이 목표이다.  
초기 능력치는 `maxHP` 10\~100, `str` 5\~30, `def` 5\~30 중 랜덤하게 주어지며, 5회까지 재시도할 수 있다.

### 이벤트
- **아무 일도 일어나지 않음**(10%)
- **배틀**(40%)
- **아이템**(20%)
- **체력 회복**(20%)  
`HP` +1 \~ +10 중 랜덤
- **랜덤 이벤트 발생**(10%)
  - mad  
`str` +3  `def` -1
  - strong  
`str` +1
  - teleport
